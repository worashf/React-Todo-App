## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

    NodeJS
    Npm

### Setup

    1. git clone git@github.com:worashf/React-Todo-App.git.
    2. cd  React-Todo-App

### Install

    npm install

### Usage

    npm  start
